其中 记忆有如下几种:
# ProceduralMemory 程序记忆 例如： 会做饭，会修车等技能 或后续一些行为习惯 例如：早上八点起床，每天洗澡，一周倒一次垃圾
# SemanticMemory 语义记忆,这里细分为放知识概念类记忆，例如: 银行可以帮你保管财产，图书馆不得大声说话，父母会在你困难时提供帮助，
# EpisodicMemory 情景记忆,例如：1月20日 去了一家餐厅吃了一顿豪华的午餐，2月20日,参加了一场盛大的晚会，2月21日 与朋友在学校发生了激烈的争吵
# OpinionMemory 观点记忆 例如：小明是一个不诚信的人, 咖啡不好喝,某家餐厅不好吃
# ProspectiveMemory 存储未来将要做的事情 例如: 后天去游乐场玩 明天有个约会 9月8日 是还款日期，10月1日 是结婚纪念日
# ShortMemory 短期记忆,通常保留一天 例如： 中午吃了一个馒头 刚才买了一个书包 与朋友之间交流说：今天的体育课很累
